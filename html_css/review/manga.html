Manga Review

読んだマンガの感想文

バカ姉弟 - 安達哲

東京の下町で生きる双子の3歳児を追ったドキュメンタリー作品。

江戸っ子風情があふれる良作品です。

アニメにもなっていたけど、雰囲気がぜんぜん伝わっていなかった。

これを読むためだけにヤンマガを買っていた時期がある。

風の谷のナウシカ - 宮崎駿

超有名なあのナウシカの漫画版

アニメージュから

映画よりも面白いけど、かなり疲れる。

あと、線が粗い？っていうのか、ペンだけで描いているみたい

7巻セットで3000円以内なのでとりあえず読めるよ

ふうらい姉妹 - 長崎ライチ

tumblrで流れてきた4コマ見て早く読みたかった

心のなかで一人ほくそ笑む感じ

なんか、この本を読んでいることを家族に知られたくない感じ

面白いよ、読んでみてね

伝染るんです。 - 吉田戦車

なんだろうね、伝説的な作品

これを読むためだけに、スピリッツを購読している友達の家に遊びに行っていた

一部キャラが独り歩きしていた気がするが、キャラ自体はそれほどではない。

1巻だけでも見てみるといい。
